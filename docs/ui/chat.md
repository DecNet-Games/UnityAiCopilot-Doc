# Chat Window
**Location**: `Tools > Unity AI CoPilot > Chat`

The central hub for coding.

## Features
*   **Context Indicator**: Shows what the AI "sees" (e.g., "Selected: Player.cs").
*   **Mode Switcher**: Toggle between `Chat`, `Code`, and `Architect` modes.
*   **Apply Button**: When code is generated, click `Apply` to write it to the file immediately.
