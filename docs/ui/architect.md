# Architect Window
**Location**: `Tools > Unity AI CoPilot > Architect`

## Features
*   **Graph View**: Visual representation of scripts and dependencies.
*   **Scan Button**: Refreshes the graph based on current codebase.
*   **Cycles Tab**: Lists any detected circular dependencies.
