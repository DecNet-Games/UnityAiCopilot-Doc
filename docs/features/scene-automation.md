# 🎬 Scene Automation

Brody is also an expert decorator. He can manipulate the Unity Hierarchy and Scene view directly.

## Powerful Use Cases
*   **Layout**: "Align all selected GameObjects in a circle with a radius of 10."
*   **Cleanup**: "Find all GameObjects with missing scripts and delete them."
*   **Material Swap**: "Find every object using the 'Dirt' material and swap it with 'Grass'."
